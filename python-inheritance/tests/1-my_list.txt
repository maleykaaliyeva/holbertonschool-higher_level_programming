The ``MyList`` class
====================

To use the ``MyList`` class, import it:
    >>> MyList = __import__('1-my_list').MyList

Create a new instance:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]

The sorted list:
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

The original list is unchanged:
    >>> print(my_list)
    [1, 4, 2, 3, 5]
